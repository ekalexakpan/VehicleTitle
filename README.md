# VehicleTitle - Vintage Vehicle Title and Registration Platform

A blockchain-based vintage vehicle title and registration platform built on Stacks, providing secure and transparent ownership records for classic and antique vehicles.

## Overview

VehicleTitle enables vintage vehicle owners to register their classic automobiles and motorcycles with detailed specifications while maintaining secure ownership records and facilitating title transfers.

## Features

- Vintage vehicle registration with VIN and condition tracking
- Vehicle category and condition assessment
- Manufacture year validation and status management
- Vehicle title transfer functionality
- Comprehensive input validation and security measures

## Smart Contract Functions

### Public Functions
- `register-vintage-vehicle`: Register vintage vehicles with detailed specifications
- `transfer-vehicle-title`: Transfer vehicle titles to new owners

### Read-Only Functions
- `get-vehicle`: Retrieve vehicle registration information
- `get-vehicle-owner`: Get vehicle owner information

## Usage

Register vintage vehicles with detailed specifications, track condition and category, and transfer titles when selling. Build a comprehensive vintage vehicle registry.

## Security

- Owner verification for vehicle title transfers
- Comprehensive input validation for all vehicle parameters
- Status-based transfer restrictions to ensure proper workflow
- VIN and description length validation for data integrity
